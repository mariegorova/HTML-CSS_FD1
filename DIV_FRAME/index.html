Бериллий - элемент главной подгруппы второй группы, второго периода периодической системы химических элементов Д.И. Менделеева, с атомным номером 4. Обозначается символом Ве (лат. Bweyllium). Высокотоксичный элемент. Прстое вещество бериллий - относительно твердый металл светло-серого цвета.
